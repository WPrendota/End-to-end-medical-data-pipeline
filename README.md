# End-to-end medical data pipeline

An end-to-end healthcare data engineering project that ingests raw hospital data, processes and transforms it into analytics-ready datasets, and delivers interactive dashboards insights.

This project builds a complete medical data pipeline using patient-level hospital records containing:

- Name
- Age
- Gender
- Blood Type
- Medical Condition
- Date of Admission
- Doctor
- Hospital
- Insurance Provider
- Billing Amount
- Room Number
- Admission Type
- Discharge Date
- Medication
- Test Results

The pipeline transforms raw healthcare data into structured datasets optimized for analytics and reporting.
